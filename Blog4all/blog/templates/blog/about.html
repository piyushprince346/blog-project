About Blog4all:

Blog4all is a blogging website. This website include blogs from several field like technology, education, sports, lifestyle, Workout, etc.
The special feature of this website is it can filter blogs according to your requirement. The only thing you have to do is to selct topic from dropdown menu on navigation bar.
This whole project(frontend and backend both) is made by Piyush B Prince.
This project uses following technology:-

  • Python
  • Django
  • Bootstrap
  • HTML
  • CSS
  • Javascript

This project also uses google fonts and Medium style Editor.
